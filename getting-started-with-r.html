1 Getting Started with R

R 是一款專為統計分析和數據視覺化設計的強大且免費的軟件環境。
它融合了統計包的便利性和編程的靈活性，廣泛應用於數據分析、機器學習和科學研究。
R 提供了豐富的統計和圖形功能，使得用戶能夠進行複雜的數據操作和高質量的圖形輸出。
以下列出 R 語言的特點：
  • 容易取得：R 軟體完全免費下載，免費使用。
  • 自由性：R 的程式語言與操作環境是非常自由的。
  • 功能性：應用廣泛且無疆界。R 的功能可以透過用戶撰寫的套件得到增強，使用者可以依照自己所需，下載相對應的套件，提高生產力。
  • 良好的線上文件說明：可以從” Help “中得到大量資訊，包含指令的使用方法、套件的使用說明…等。
  • 高階程式語言：其計算、分析的速度不亞於其他專業軟體。且對於學過 C、Java…等語言的使用者，很容易上手。
  • 繪圖能力：R 軟體擁有強大的繪圖能力，也可以透過不同的套件，增強圖形視覺化的程度，讓視覺化的圖形代替數字說話，增加說服力。

    1.3 基本操作

    在免費電子書 – R 統計軟體有提供完整的向量、矩陣、函數等基本操作介紹，這裡舉幾個基本的例子

    # 定義向量 x 為 2 到 9
x <- 2:9
# 看看 x 長什麼樣子
x
    ## [1] 2 3 4 5 6 7 8 9
    # 向量每個元素 +1
x + 1
    ## [1]  3  4  5  6  7  8  9 10
    # 向量每個元素 *5
x * 5
    ## [1] 10 15 20 25 30 35 40 45
    # 定義 y 為 x 的 2 倍
y <- 2 * x
# 另一種建向量方式
z <- c(1, 2, 3, 4)
# 向量也可以放字串
x <- c(3, "abc", 4)
x
    ## [1] "3"   "abc" "4"
    # 再改回純數值向量
x <- c(3, 5, 9)
# 抓第 2 個元素
x[2]
    ## [1] 5
    # 同時抓第 1 和第 3 個元素
x[c(1, 3)]
    ## [1] 3 9
    # 抓第 1 到第 2 個元素
x[1:2]
    ## [1] 3 5

    1.4 資料型態

    R 的資料型態可用 class() 或 str() 或 View() 查詢。以下介紹五種主要型態：

    1.4.1 Character：文字字串

    文字字串必須用雙引號或單引號包起來。

    # 定義為字串
x <- "Hello world."
x
    ## [1] "Hello world."
    class(x)
    ## [1] "character"

    1.4.2 Numeric：數值型態

    Numeric 可以用於各種運算：加、減、乘、除、取餘數等。

    # 定義 numeric
x <- 100
class(x)
    ## [1] "numeric"
    # 四則運算與餘數
x + 5
    ## [1] 105
    x - 5
    ## [1] 95
    x * 5
    ## [1] 500
    x / 3
    ## [1] 33.33333
    x %% 3
    ## [1] 1
    # sum() 可對向量求和
v <- c(10, 11, 12, 11)
sum(v)
    ## [1] 44

    1.4.3 Integer：整數型態

    預設的數字是 numeric，若要指定為整數可用 as.integer()。

    # 定義為整數
x <- as.integer(3)
class(x)
    ## [1] "integer"
    # 小數轉整數會捨去小數部分
y <- as.integer(3.5)
y
    ## [1] 3
    class(y)
    ## [1] "integer"

    1.4.4 Complex：複數型態

    複數由實部與虛部組成，虛部以 i 表示。

    # 定義為整數
x <- as.integer(3)
class(x)
    ## [1] "integer"
    # 小數轉整數會捨去小數部分
y <- as.integer(3.5)
y
    ## [1] 3
    class(y)
    ## [1] "integer"

    1.4.5 Logical：邏輯值

    邏輯值只有 TRUE 或 FALSE，可簡寫為 T、F。

    # 定義邏輯值
x <- TRUE
y <- FALSE
class(x)
    ## [1] "logical"
    class(y)
    ## [1] "logical"

    1.5 資料型態的轉換

    使用 Y <- as.資料型態(X) 可以將 X 轉換成不同的資料型態，並將結果指派給 Y。

    # 將 numeric 轉成 character
x <- 3
x <- as.character(x)
class(x)
    ## [1] "character"
    # 將 numeric 轉成 factor
y <- 3
y <- as.factor(y)
class(y)
    ## [1] "factor"

    1.6 其他小功能

    1. 可以善用註記功能：# ，程式不會編譯由 # 開頭的那一行
    2. help 查詢資訊
    # ?? 可以全文搜尋 ggplot 相關主題
??ggplot 
    ## 開啟 httpd 求助伺服器… 好了
    # ? 跟 help()，這兩個指令的作用相同可以查詢單一函式文件
?c
help(c)
    1. 避免中文路徑與檔名：請使用英文命名檔案、資料夾、變數和使用者名稱，以減少編碼相容性問題。
    2. 先檢查資料型態：讀入資料後，務必以 class() 或 str() 確認欄位屬性正確，再進行後續分析。
    3. 注意大小寫敏感：R 對大小寫非常嚴格，如遇「找不到物件」的錯誤，請先檢查拼寫與大小寫是否一致。
    4. 命名要有意義：在處理大型資料或研究專案時，具描述性的變數名稱能快速辨識欄位並降低出錯機率。
    5. Warning 不必驚慌：只要不是 error，就可以先忽略，warning 僅提醒可能的潛在問題。
