11.2 2.適配度檢驗(Goodness-of-Fit Test)

使用範例 12.1 來進行適配度檢驗，檢查觀察到的數據是否符合假設的分布。

11.2.0.1 I. 輸入數據、假設

# 輸入觀察到的客戶數量
number.customers <- c(70, 60, 54, 16)

# 輸入市場份額分布
market.share <- c(0.40, 0.32, 0.24, 0.04)
H_0: p_1 = 0.40, p_2 = 0.32, p_3 = 0.24, p_4 = 0.04
H_A: At least one p_i differs from its hypothesized value.

11.2.0.2 II. 設定 α 值

α = 0.05
11.2.0.2.1 III. 進行卡方適配度檢驗
  1. 準備數據: 我們已經在上一部分中準備好了 number.customers 和 market.share 兩個向量。
  2. 運行檢驗: 在這裡，我們將 number.customers 作為第一個參數，並將 p 設置為 market.share，這樣 R 會將這些預期概率用於檢驗。
# 進行卡方適配度檢驗
chisq.test.result <- chisq.test(number.customers, p = market.share)

# 顯示結果
print(chisq.test.result)
## 
## Results of Hypothesis Test
## --------------------------
## 
## Alternative Hypothesis:          
## 
## Test Name:                       Chi-squared test for given probabilities
## 
## Data:                            number.customers
## 
## Test Statistic:                  X-squared = 10.25
## 
## Test Statistic Parameter:        df = 3
## 
## P-value:                         0.01655632

11.2.0.3 IV.是否拒絕 H_0

P-value: 0.01655632
p 值 < 0.05，拒絕 H_0

11.2.0.4 V. 結論

At least one p_i differs from its hypothesized value.

11.3 3.獨立性檢驗(Test for Independence)

基於表 12.8 中的數據，使用卡方檢驗來檢查性別和車輛類型之間是否存在獨立性。

11.3.0.1 I. 構建列聯表(contingency table)、假設

# 輸入數據
female <- c(50, 60)
male <- c(120, 170)

# 創建列聯表(contingency table)
table <- rbind(female, male)

# 為列聯表命名列
colnames(table) <- c("economy.car", "noneconomy.car")

# 顯示列聯表
print(table)
##        economy.car noneconomy.car
## female          50             60
## male           120            170
H_0(虛無假設):  性別和車輛類型之間是獨立的
H_A(對立假設) : 性別和車輛類型之間存在關聯（即不獨立）

11.3.0.2 II. 設定 α 值

α = 0.05

11.3.0.3 III. 進行卡方獨立性檢驗

# 進行卡方獨立性檢驗
chisq.test.result <- chisq.test(table, correct = FALSE)

# 顯示結果
print(chisq.test.result)
## 
## Results of Hypothesis Test
## --------------------------
## 
## Alternative Hypothesis:          
## 
## Test Name:                       Pearson's Chi-squared test
## 
## Data:                            table
## 
## Test Statistic:                  X-squared = 0.541975
## 
## Test Statistic Parameter:        df = 1
## 
## P-value:                         0.4616154

11.3.0.4 IV.是否拒絕 H_0

P-value: 0.4616154
p 值 ≥ 0.05，則無法拒絕 H_0，表示目前的數據不足以支持性別和車輛類型之間存在關聯。

11.3.0.5 V. 結論

我們無法拒絕 H_0，沒有足夠的證據表明性別和車輛類型之間存在關聯。

11.4 4. 常態性檢驗(Tests for Normality)

11.4.0.1 I. 假設

H_0(虛無假設):  數據服從常態分布
H_A(對立假設) : 數據不服從常態分布

11.4.0.2 II. 設定 α 值

α = 0.05

11.4.0.3 III. 使用 Pearson 卡方檢驗進行常態性檢驗

# 使用 nortest 包中的 pearson.test() 函數來進行常態性檢驗
pearson.test(household.income.df[, "Household.Income"])
## 
##  Pearson chi-square normality test
## 
## data:  household.income.df[, "Household.Income"]
## P = 22, p-value = 0.00254
11.4.0.3.1 結果解讀
檢驗名稱: Pearson 卡方常態性檢驗
檢驗統計量: P = 22
p 值: 0.002540414

根據 p 值（0.002540414），我們拒絕 H_0，表示數據不服從常態分布。

11.4.0.4 IV. 使用 Jarque-Bera 檢驗進行常態性檢驗

# 使用 fBasics 包中的 jarqueberaTest() 函數來進行常態性檢驗
jarqueberaTest(household.income.df[, "Household.Income"])
## 
## Title:
##  Jarque-Bera Normality Test
## 
## Test Results:
##   STATISTIC:
##     X-squared: 116.1437
##   P VALUE:
##     Asymptotic p Value: < 2.2e-16
11.4.0.4.1 結果解讀
檢驗名稱: Jarque-Bera 常態性檢驗
檢驗統計量: X-squared = 116.1437
p 值: < 2.2e-16

根據 p 值（< 2.2e-16），我們拒絕 H_0，表示數據不服從常態分布。

11.4.0.5 V. 使用 Anderson-Darling 檢驗進行常態性檢驗

# 使用 nortest 包中的 ad.test() 函數來進行常態性檢驗
ad.test(household.income.df[, "Household.Income"])
## 
##  Anderson-Darling normality test
## 
## data:  household.income.df[, "Household.Income"]
## A = 2.9251, p-value = 1.842e-07
11.4.0.5.1 結果解讀
檢驗名稱: Anderson-Darling 常態性檢驗
檢驗統計量: A = 2.925133
p 值: 1.842161e-07

根據 p 值（1.842161e-07），我們拒絕 H_0，表示數據不服從常態分布。

11.4.0.6 VI. 結論

根據以上三種常態性檢驗結果，我們均拒絕 H_0，表示 Household.Income 數據不服從常態分布。
無論使用哪一種檢驗方法，結論都是一致的：該數據集不符合常態分布。
